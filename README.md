# Smart Email Writer

Smart Email Writer is an AI-powered application that generates professional email replies and integrates directly inside Gmail using a Chrome Extension.  
It helps users save time and improve email productivity by generating context-aware responses instantly.

---

## Technology Used
- React (Frontend)
- Spring Boot (Backend)
- JavaScript (Chrome Extension)
- REST APIs
- AI API (Gemini / OpenAI)
